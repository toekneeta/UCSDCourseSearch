Hello, this website is designed to be a semantic search for UCSD classes! Feel free to search for any topics you're interested in and further refine your searches with our filters!

Here's some tips on using our search:

  • Highlighted courses are offered Spring 2024.
  • For each search result, its CAPES can be found by clicking the course code.
  • Rate the relevance of search results with the thumbs up/down buttons.
  • Filtering by departments is case-insensitive (just make sure department codes are spelled correctly and separated by commas).
  • You can search by course code (but results will only be returned for exact matches)

Things to come:

  • General UI improvements.

Disclaimer: The course availability on this website may not reflect all courses offered at UCSD. If there are any discrepancies in course offerings (or for any general feedback), please let us know in Feedback!
